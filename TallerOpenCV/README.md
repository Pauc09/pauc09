# Taller OpenCV - Procesamiento de Imágenes

## Ejercicios Implementados

### Punto 1: Conversión RGB → HSV
Implementación manual de la conversión de espacio de color RGB a HSV.

### Punto 2: Modificar Saturación
Aumentar la saturación de una imagen trabajando con valores HSV.

### Punto 3: K-Means Manual
Implementación completa del algoritmo K-Means para cuantización de colores.

### Punto 4: Gray World
Algoritmo de constancia de color para corrección de balance de blancos.

### Punto 6: Corrección Gamma
Ajuste de brillo y contraste mediante corrección gamma.

### Punto 7: Corrección de Viñeteo
Compensación del oscurecimiento en las esquinas de la imagen.

## Compilar y Ejecutar
```bash
cd puntoX
mkdir build && cd build
cmake ..
make
./puntoX
```

## Requisitos
- OpenCV 4.x
- CMake 3.10+
- C++11
